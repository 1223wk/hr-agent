## HR与行政共享服务

## 业务背景：
企业内部HR与行政规章制度体系庞大、繁琐，涵盖考勤、年假、差旅、福利等内容，同时制度标准随员工职级、属地、工龄等动态变化，同一问题对不同员工的适配规则完全不同。

## 项目边界：
项目范围内：内容涵盖企业员工手册，支持员工咨询、解答；实现24小时智能自助回答，缺失身份主动追问，答案合规检验
项目范围外：不涉及核心人事业务告知；不支持薪资谈判等非常规问题咨询；不代替（如报销）高危、重大人事业务的人工审批与复核，无审批等流程发起权限。

## 重难点：
企业政策存在职级、工龄、属地等多维度交叉条件，普通检索无法精准区分差异，容易出现错配、漏配问题。
员工提问口语化、表述不规范，关键词匹配干扰项多，语义匹配难度大。

## 风险点：
身份匹配错误；知识库内容未同步更新；答案漏召回、回复不全；模型擅自解读产生幻觉；人员隐私泄露

## 核心模块：
用户身份管理模块、RAG模块、Agent问答生成模块、审计管控模块
